Privacy Policy

LAST UPDATED — JANUARY 2026

Information We Collect

Vox collects the minimum data required to function: server IDs, channel and role configuration, command usage metadata, and moderation logs (such as warnings or timed actions). Message content is processed only when directly relevant to a command or AI request, and is not stored beyond that processing.

As a development build, Vox may also log technical diagnostics — such as error traces and command performance — to help identify and fix issues during testing.

How Information Is Used

Collected data is used strictly to operate Vox's features — running commands, enforcing moderation settings, generating AI responses, and improving stability during this testing phase. Data is never sold, and is not used for advertising.

Data Storage

Server configuration and moderation records are stored on secured infrastructure with restricted access. Data is retained only for as long as Vox remains active in a server; removing Vox from a server triggers deletion of its associated configuration data within a reasonable period.

Data Sharing

Vox does not share, sell, or rent collected data to third parties. Limited technical data may be processed by the infrastructure providers that host Vox, solely to keep the service running, and only under agreements that restrict its use to that purpose.

User Rights

  • Server administrators can request an export of their server's stored configuration and logs.
  • Any user can request deletion of data tied to their individual user ID across all servers.
  • Removing Vox from a server revokes its access to that server's data going forward.

Security

Reasonable technical and organizational measures are used to protect stored data against unauthorized access, alteration, or loss. As Vox is an active development build, security practices continue to be reviewed and improved as the project matures.

Policy Updates

This Privacy Policy may be revised as Vox evolves, particularly during this development phase. Material changes will be reflected on this page with an updated revision date.

Contact

For data requests or privacy questions, reach out to knife4love via the support server.

Vox Command Manual

Learn how to invite Vox, use commands, manage permissions, and follow upcoming features.

Welcome

Vox is an advanced AI-powered Discord assistant built to assist, automate, and evolve. It is designed for clean interactions, useful commands, and a premium server experience.

Getting Started

1. Invite Vox using the Invite button.

2. Select a Discord server where you have Manage Server permission.

3. Authorize Vox.

4. Type /help to view commands.

Commands

Utility

/ping — Displays Vox’s latency.

/help — Shows available commands.

/about — Shows information about Vox.

Information

/userinfo — Displays information about a Discord user.

/serverinfo — Displays information about the current server.

Communication

/say — Sends a message through Vox using an embed.

Permissions

Vox may need these permissions:

• Send Messages

• Use Slash Commands

• Embed Links

• Read Message History

• View Channels

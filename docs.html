EVERYTHING NEEDED TO SET UP AND RUN VOX

Welcome

Vox is an advanced, AI-powered assistant built for Discord — designed to assist, automate, and evolve alongside the servers that use it. This page covers everything needed to get Vox running in a server, from initial setup to the full command reference.

SETUP

Getting started

Vox takes under a minute to set up. No configuration is required to begin.

1. Invite Vox

Use the Invite Vox link at the top of this page, or the button on the homepage, to open Discord's authorization screen.

2. Add Vox to a server

Select the server you'd like to add Vox to from the dropdown, review the requested permissions, and confirm. You'll need the "Manage Server" permission on the target server to complete this step.

3. Use /help

Once Vox has joined, type /help in any channel it can see. This brings up the full list of available commands, grouped by category, directly inside Discord.

4. Begin using commands

Vox's commands are all slash commands — type / in the message box and select Vox from the list to see suggestions as you type. No prefixes or special syntax to memorize.

REFERENCE

Commands

The current command set, grouped by category. New commands are added as Vox develops.

Utility

  • /ping — Displays the bot latency.

  • /help — Displays all available commands.

  • /about — Information about Vox.

Information

  • /userinfo — Displays information about a Discord user.

  • /serverinfo — Displays information about the current server.

Communication

  • /say — Sends a message through Vox using an embed.

SETUP

Permissions

To function correctly, Vox requires the following permissions when invited to a server:

  • Send Messages
  • Use Slash Commands
  • Embed Links
  • Read Message History
  • View Channels

These permissions are requested automatically as part of the invite link — no manual configuration is needed.
